Code the App - 1

Import the guizero library

To create an app, begin by importing the guizero module of code. For this app you will need classes (which are templates) to make an app and to produce text and pushbuttons.

Above the line where you imported the random module, enter

from guizero import App, Text, PushButton

As these are classes, they begin with upper case letters.

Below the list of puns, delete the line that choose a pun at random and display it on screen.
