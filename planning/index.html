Planning

Before you start to code, do a little planning.

Where to save your app

It's best to make a new directory for your app. Decide on a name, for example, mypuns.

What to call it

Next, decide what to call your Python script, for example, suzyspuns.

Remember, stick to lower case and no spaces! You don't need the .py extension if you are working in Linux. Just remember to add the shebang as the first line so that Linux knows that you want to run your file using Python.

Now decide on a name for your app. app is convenient; you will find this name used in many examples.

Please make a note of what you decide - paper and pen(cil) or a doc. This will help you avoid confusion.

When you have finished, move to the section entitled Getting Started.
